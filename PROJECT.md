# Student Database Management System (C)

A simple menu-driven student database management system written in C using structures and file handling.

## Features
- Add student records
- Display all students
- Search student by roll number
- Delete student records

## Technologies Used
- C Programming
- File Handling
- Structures

## How to Run
1. Compile the program using a C compiler
2. Run the executable file
3. Use the menu to manage student records
